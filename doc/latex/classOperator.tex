\doxysubsection{Detailed Description}
The \doxylink{classOperator}{Operator} class provides a generic operator up to three-\/body, scalar or tensor. The class contains lots of methods and overloaded operators so that the resulting code that uses the operators can look as close as possible to the math that is written down.
